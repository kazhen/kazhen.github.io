Data & Web Technologies for Data Analysis

About This Class

This course focuses on the fundamental concepts of data technologies used in data analysis. Specifically, we mainly used Python and Jupyter Notebook. Our analysis used data from traditional databases and Web APIs. Please see some of the projects I did below.

Projects

  • Doomsday Algorithm

    • Doomsday Algorithm was implemented in Python to count the number of Friday the 13's and to analyze the occurances of birthdays in 1978

    See the algorithm and analysis here.

  • Image Processing Basics and Singular Value Decomposition

    • Given an image, the color channels were manipulated to show different colors. Then I implemented singular value decomposition to reduce the number of bits as well as maintaining visibility.

    An example of a cat image manipulation is shown here.

    • A various amount of fruits and vegetables were saved in Excel files. After reading each file in and extracted neccessary data, I performed analysis on the food such as finding the most and least expensive fruit.
