STA 141A

Fundamentals of Statistical Data Science

About This Class

STA 141A is an introduction to data analysis and visualization using R. The data analysis included clean and complex data. With complex data, we cleaned the data using regular expressions. Some of the projects from this course can be seen below.

Projects

  • Linear and Logistic Regression in R

    • Given a dataset, I performed bootstrap linear regression to find the best model. Using bootstrap helps creates better results as we'e randomly sampling multiple times. Logistic regression was performed a dataset for flowers to see how well the model performs. We split the data into testing and training set to see how well the model classifies certain flower data.

    See the prediction here.

  • Regular Expressions and Webscraping

    • Given a set of files on car data, I used regular expressions to compile an excel with concise data such as car model, car color, VIN, location it was sold, etc. In addition, I also webscraped Professor Mueller's website. We compiled a dataframe with all the authors, title of publication, year, and more.

    See the regular expression and web scrapping techniques here.

  • Mapping of Data on University of California Campuses

    • With the University of California schools, I used Rgooglemaps, maps, and ggmap packages to get the distance apart the campuses are and plotted bike routes between the campuses.

    See the map here.
