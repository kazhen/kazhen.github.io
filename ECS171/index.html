About This Class

This was an introduction to Machine Learning where the course covered supervised and unsupervised learning, including classification, dimensionality reduction, regression and clustering using modern machine learning methods.

Projects

Since baseball has a lot of predictors, we first found the best model and did Principal Component Analysis. Furthermore, we did classification using Nuetral Networks.
